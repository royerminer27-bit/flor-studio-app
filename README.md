# 🌸 Flor Studio

## Descripción

Flor Studio es una aplicación web progresiva (PWA) diseñada para administrar un negocio de flores artesanales.

## Objetivo

La aplicación permitirá gestionar:

- 📦 Inventario
- 💰 Costos
- 🌹 Productos
- 🏭 Producción
- 👥 Clientes
- 🛒 Ventas
- 📊 Reportes
- 📷 Catálogo de productos
