# 🌸 Flor Studio - Requisitos Funcionales

## Introducción

Este documento define las funciones principales que tendrá la aplicación Flor Studio.

Los requisitos describen qué debe hacer el sistema y cómo interactuará el usuario.

---

# RF-01 Gestión de usuarios

El sistema permitirá:

- Crear usuarios.
- Iniciar sesión.
- Cerrar sesión.
- Administrar permisos.

Usuarios iniciales:

- Cuchurrumi.
- Araguatita.

---

# RF-02 Dashboard principal

El sistema mostrará un panel con información importante:

- Ventas del día.
- Ventas mensuales.
- Ganancias.
- Pedidos pendientes.
- Inventario bajo.
- Productos más vendidos.

---

# RF-03 Gestión de materiales

El usuario podrá:

- Registrar materiales.
- Editar materiales.
- Eliminar materiales.
- Consultar existencias.

Cada material tendrá:

- Nombre.
- Categoría.
- Color.
- Imagen.
- Proveedor.
- Costo.
- Unidad de medida.
- Stock actual.
- Stock mínimo.

---

# RF-04 Control de inventario

El sistema permitirá registrar:

## Entradas

Ejemplo:

Compra de 10 paquetes de limpiapipas.

## Salidas

Ejemplo:

Fabricación de 20 rosas.

El inventario se actualizará automáticamente.

---

# RF-05 Gestión de productos

El usuario podrá crear productos:

Ejemplos:

- Rosa.
- Tulipán.
- Girasol.
- Ramos.
- Cuadros.
- Lámparas.

Cada producto tendrá:

- Nombre.
- Categoría.
- Imagen.
- Precio.
- Tiempo de fabricación.

---

# RF-06 Recetas de fabricación

Cada producto podrá tener una receta.

La receta incluirá:

- Material utilizado.
- Cantidad necesaria.
- Unidad.
- Tiempo de elaboración.

El sistema calculará automáticamente:

- Costo del producto.
- Material necesario.

---

# RF-07 Cálculo de costos

El sistema calculará:

Costo materiales

+

Costo mano de obra

+

Gastos indirectos

=

Costo real


También permitirá definir:

- Margen de ganancia.
- Precio recomendado.

---

# RF-08 Gestión de pedidos

El usuario podrá:

- Crear pedidos.
- Asociar clientes.
- Definir fecha de entrega.
- Cambiar estado del pedido.

Estados:

- Pendiente.
- En producción.
- Terminado.
- Entregado.

---

# RF-09 Producción

El sistema calculará:

- Material necesario.
- Tiempo requerido.
- Fecha estimada.
- Ganancia esperada.

---

# RF-10 Clientes

El sistema permitirá:

Registrar:

- Nombre.
- Teléfono.
- WhatsApp.
- Dirección.
- Historial de compras.

---

# RF-11 Ventas

Permitirá registrar:

- Producto vendido.
- Cantidad.
- Cliente.
- Precio.
- Método de pago.
- Ganancia.

---

# RF-12 Reportes

El sistema generará información sobre:

- Ventas.
- Ganancias.
- Productos.
- Materiales.
- Inventario.
