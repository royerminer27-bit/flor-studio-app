# 🌸 Flor Studio - Roadmap de Desarrollo

## Introducción

Este documento define las fases de desarrollo de Flor Studio.

El objetivo es construir la aplicación progresivamente, comenzando por las funciones esenciales y agregando funcionalidades avanzadas posteriormente.

---

# Fase 1 - Núcleo del sistema (MVP)

Objetivo:

Crear una primera versión funcional para administrar costos e inventario.

## Funciones:

### Usuarios

- Registro de usuarios.
- Inicio de sesión.
- Gestión de permisos.

### Materiales

- Crear materiales.
- Editar materiales.
- Eliminar materiales.
- Control de stock.

### Productos

- Crear productos.
- Categorías.
- Imágenes.
- Precio de venta.

### Recetas

- Asociar materiales a productos.
- Definir cantidades.
- Calcular costo automático.

### Calculadora de costos

Calcular:

Materiales
+
Mano de obra
+
Gastos indirectos

=

Costo real

---

# Fase 2 - Gestión del negocio

Objetivo:

Controlar la operación diaria del taller.

## Funciones:

### Compras

- Registrar compras.
- Asociar proveedores.
- Actualizar inventario.

### Proveedores

- Información de contacto.
- Historial de compras.

### Clientes

- Registro de clientes.
- Historial de pedidos.

### Pedidos

- Crear pedidos.
- Fechas de entrega.
- Estados.

---

# Fase 3 - Producción inteligente

Objetivo:

Organizar la fabricación.

## Funciones:

- Agenda de producción.
- Planificación automática.
- Cálculo de materiales necesarios.
- Tiempo estimado.
- Control de producción.

Ejemplo:

Pedido:

100 rosas

Resultado:

Material necesario:
- Limpiapipas.
- Pistillos.
- Floral tape.

Tiempo:
34 horas.

---

# Fase 4 - Reportes e inteligencia

Objetivo:

Ayudar a tomar decisiones.

Funciones:

- Dashboard avanzado.
- Gráficas de ventas.
- Productos más rentables.
- Control de ganancias.
- Alertas de inventario.

---

# Fase 5 - Expansión

Funciones futuras:

## Catálogo

- Página pública.
- Compartir productos.
- Enlaces.

## WhatsApp

- Enviar catálogos.
- Confirmar pedidos.

## Flora

Asistente del sistema.

Ejemplos:

"El stock de pistillos está bajo."

"Hoy debes fabricar 15 rosas."

"Este producto tiene mayor rentabilidad."

---

# Objetivo final

Convertir Flor Studio en una plataforma especializada para pequeños negocios artesanales.

Un sistema que permita crear productos, controlar costos, administrar producción y aumentar la rentabilidad.
